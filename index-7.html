PROJ_01 Infrastructure Windows Server — AD, DNS, DHCP
Windows Server 2022Active DirectoryDHCP
▾
Contexte

L'entreprise fictive BioVision (recherche biomédicale) souhaite centraliser ses comptes utilisateurs et automatiser l'attribution des adresses IP. Ma mission : déployer une infrastructure Windows Server avec AD, DNS et DHCP.

Objectifs
01Déployer Windows Server 2022 avec IP statique (172.20.0.10)
02Créer un domaine Active Directory (biovision.com)
03Mettre en place DNS + DHCP avec étendues par VLAN
04Intégrer un poste client Windows 11 au domaine
Travail réalisé
→Installation WS2022 + promotion contrôleur de domaine
→Configuration DHCP avec étendues VLAN, intégration Windows 11
Compétences mobilisées
A1.3.1 — Installation serveur
A1.3.4 — Annuaire AD
A1.3.5 — Service DHCP
A4.1.2 — Intégration domaine
A4.1.3 — Tests et validation réseau

PROJ_03 Réseau sécurisé — VLAN, DMZ & Routage inter-VLAN
CiscoDMZVLANACL
▾
Contexte

BioVision souhaite isoler ses services sensibles (Administration, Vente, R&D…) et héberger un service web accessible depuis l'extérieur via une DMZ sécurisée.

VLAN déployés
10Administration
21Ventes
22Marketing
23R&D
30Wifi
31DMZ
Travail réalisé
01Analyse — identification services à isoler, besoins d'accès inter-VLAN
02VLAN — switch L3, sous-réseaux et plages DHCP pour chaque VLAN
03DMZ — VLAN 31, HTTP/HTTPS uniquement, blocage accès internes
04Routage inter-VLAN — interfaces SVI, règles ACL sur switch L3
05Tests & validation — communication VLAN, isolation DMZ, DHCP
